# FareCalc Travel Optimizer

## Overview
A Python script that calculates ride fare based on distance, vehicle type, and peak hour surge pricing.

## Features
- Calculates fare using distance and vehicle rates
- Applies 1.5x surge during peak hours (17–20)
- Handles invalid vehicle types
- Displays a formatted ride estimate
- Basic error handling for invalid input

## Rates per KM
- Economy: 10
- Premium: 18
- SUV: 25

## Concepts Used
- Dictionary
- Functions
- Conditional statements
- Exception handling
